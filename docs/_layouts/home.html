How to use this course

  1. Download the working file linked at the top of each module.
  2. Read & do — open the file in Excel and try every example as you go.
  3. Quiz — take the 5–7 question check at the end. Instant feedback in your browser.
  4. Compare with the solution file when you're done.
  5. Capstone — after Module 5, build the dashboard from scratch.

Targeted at Microsoft 365 / Excel 2021+. No macros, no Power Query — just the 80% of Excel you'll actually use day to day.
