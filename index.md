# Excel 101 for Sales

A self-paced **4-hour** Excel course built for **sales professionals** who already know the basics and want to confidently work with sales data.

By the end of the course you will be able to:

- Structure messy sales data into clean, filterable tables
- Use modern lookup and aggregation formulas (XLOOKUP, SUMIFS, dynamic arrays)
- Build pivot tables and pivot charts that answer real sales questions
- Visualize sales performance with clear charts and conditional formatting
- Validate inputs, run quick what-if analysis, and put it all together in a small dashboard

> Targeted at **Microsoft 365 / Excel 2021+**. A short note covers VLOOKUP for backwards compatibility, but the course teaches the modern functions first.

## Course outline (≈ 4 hours)

| # | Module | Time | What you will do |
|---|--------|------|------------------|
| 1 | [Working with sales data](lessons/01-tables-and-data) | 35 min | Tables, structured refs, sorting, filtering, basic cleanup |
| 2 | [Sales-power formulas](lessons/02-formulas) | 75 min | IF / IFS, XLOOKUP, SUMIFS family, date and text functions, dynamic arrays |
| 3 | [Pivot tables & pivot charts](lessons/03-pivot-tables) | 60 min | Group, slice, summarise sales by region / rep / product / period |
| 4 | [Visualizing sales data](lessons/04-visualization) | 30 min | Right chart for the job, conditional formatting, sparklines |
| 5 | [Validation, what-if & capstone](lessons/05-validation-and-capstone) | 40 min | Drop-downs, Goal Seek, named ranges + a small dashboard project |

Each module ends with a short [quiz](quizzes/) and a downloadable working file. Solution files are in [files/solutions/](https://github.com/scripts-and-tables/excel-101/tree/main/files/solutions).

## How to use this course

1. **Download the dataset** (linked from each module) and open the working file in Excel.
2. **Read the lesson** alongside Excel — try every example as you go.
3. **Take the mini-quiz** at the end of the module (instant feedback, nothing is sent anywhere).
4. **Compare your file with the solution file** when you finish.
5. After Module 5, build the **capstone dashboard** from scratch.

You can do the modules in order (recommended) or jump to a specific topic.
